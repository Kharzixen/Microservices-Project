# Cart service

Cart service for the microservices project using Spring-boot and MongoDB

Implemented functionalities: 
    Create blank cart
    Add item to this cart

Modules:
    Mapper: Mapstruct
    Spring Data Mongodb. 

## Developer notes:

    Creation of cart and item creation for that cart is implemented
